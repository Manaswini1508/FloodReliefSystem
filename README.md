# Flood Relief System

## Overview

This project is an ML-based disaster management system that predicts disaster severity and helps in efficient resource allocation.

##  Features

* Predicts disaster severity using Machine Learning (Decision Tree)
* Flask backend API
* Interactive frontend (HTML, CSS, JavaScript)
* Stores disaster data using SQLite database

## 🛠 Tech Stack

* Python
* Flask
* SQLite
* HTML, CSS, JavaScript
* Scikit-learn

##  Project Structure

* backend/ → Flask API and ML model
* frontend/ → User Interface

## ▶️ How to Run

1. Go to backend folder
2. Install required libraries
3. Run:
   python app.py
4. Open frontend/index.html in browser

##  Real-Time Use Case

Used by disaster management teams to quickly assess severity and allocate resources like food, water, and rescue teams efficiently.
